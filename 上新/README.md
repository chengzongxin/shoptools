# 网页自动化项目

这是一个基于 Selenium 的网页自动化项目，用于模拟用户操作。

## 环境要求
- Python 3.8+
- Chrome 浏览器

## 安装步骤

1. 安装依赖包：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

## 项目结构
- `main.py`: 主程序文件
- `config.py`: 配置文件
- `requirements.txt`: 项目依赖文件
